# Random Password Generator
**Автор:** [Шурупова Ксения Игоревна]

## Описание
GUI-приложение на Python для генерации надежных случайных паролей. 
Позволяет настраивать длину и используемые наборы символов.

## Функционал
- Настройка длины (от 4 до 32 символов).
- Выбор типов символов (цифры, буквы, знаки пунктуации).
- Просмотр истории последних генераций.
- Автоматическое сохранение истории в JSON-файл.

## Примеры использования (Тесты)
1. Установите длину 12, включите только "Буквы" -> Нажмите "Сгенерировать".
2. Включите все чекбоксы -> Пароль должен содержать символы разных типов.
3. Перезапустите программу -> История в таблице должна восстановиться из файла.итоговая аттестация
